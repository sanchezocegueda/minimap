# Getting started with development on a fresh machine

## Setup repo
```bash
# ssh
git clone git@github.com:sanchezocegueda/minimap.git
# http
git clone https://github.com/sanchezocegueda/minimap.git
cd minimap
git submodule update --init --recursive
```
